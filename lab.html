Laboratories


Slno Laboratory Used for/Uses
1 Software Development Laboratory To develop simple software applications using C,C++ and Java
2 Free / Open Source Software Laboratory To familiarize students with the various Open source software like Linux,NetworkSimulators,Mysql,Eclipse and Glomosim.
3 Internet Programming Laboratory To use the freely available open source tools like XAMPP/WAMP Servers.
4 RDBMS Laboratory Used by students to create and interact with the database.
5 SYSTEM SOFTWARE DEVELOPMENT LABORATORY To familiarize students with the various system software like assemblers,loaders,linkers,macro processors and text editor.
6 NETWORK LABORATORY To train the students in developing network applications like socket creation, client/server applications etc.
7 Multimedia Lab & PG Laboratory
  • To perform image editing operations using Photoshop Tool.
  • To do simple 2D animation operations using "Flash" Tool.
  • To do 3D animation using 3Dmax software.
8 Cloud Computing Laboratory This lab is utilized by the students to learn and develop cloud environment using cloud sim.
9 Mobile Application Development Lab To train the students in developing the mobile applications.
10 Mobile Communication Laboratory This lab provides a platform for developing IoT applications and working with mobile communication related experiments,
